An **instance** declares that a specific type satisfies a type class.

````lean
structure OfficeBuilding where
  campus : String
  name : String
  visitorAddress : String
  postAddress : String
  numFloors : Nat
  numOffices : Nat
  parkingSpaces : Nat

instance : Building OfficeBuilding where
  campus b   := b.campus
  name b     := b.name
  address b  := b.visitorAddress   -- can map fields freely
  numFloors b := b.numFloors
````


````lean
structure LectureBuilding where
  campus : String
  name : String
  address : String
  numFloors : Nat
  numLectureHalls : Nat
  hasCafeteria : Bool

instance : Building LectureBuilding where
  campus b   := b.campus
  name b     := s! "{b.campus}:{b.name}"  -- can compute derived values
  address b  := b.address
  numFloors b := b.numFloors
````

Instances can **transform or compute** values — they're not limited to simple field access.
