## The Idea

Instead of building custom combined monads, **wrap one monad inside another** using a standardized pattern. A **monad transformer** takes an existing monad `m` and adds one new effect to it.

## StateT — Adds State to Any Monad

```lean
structure StateT (σ : Type) (m : Type → Type) (α : Type) where
  run : σ → m (α × σ)
```

Compare with plain `StateM`:

```
StateM σ α  =  σ → (α × σ)          -- result is a bare pair
StateT σ m α  =  σ → m (α × σ)      -- result is wrapped in monad m
```

If `m = Id` (identity monad), you get `StateM` back. If `m = Except ε`, you get state + errors. If `m = IO`, you get state + I/O.

## ExceptT — Adds Errors to Any Monad

```lean
structure ExceptT (ε : Type) (m : Type → Type) (α : Type) where
  run : m (Except ε α)
```

The inner monad `m` produces an `Except` result — either success or failure.

## Monad Instances

Each transformer defines a `Monad` instance that delegates to the wrapped monad's `pure` and `bind`:

```lean
instance [Monad m] : Monad (StateT σ m) where
  pure a := ⟨fun s => pure (a, s)⟩            -- 'pure' here is m's pure
  bind f g := ⟨fun s => do                    -- 'do' here uses m's bind
    let (a, s') ← f.run s
    (g a).run s'⟩
```

The pattern: the transformer handles its own effect (state threading, error propagation) and delegates everything else to `m`.

## Using Transformers

Define your combined monad as a **stack** of transformers with a type alias:

```lean
abbrev M := StateT Int (Except String)        -- state + errors
abbrev M := StateT Int (ExceptT String IO)    -- state + errors + I/O
```

Then use standard operations — `throw`, `modify`, `IO.println` — they work through the stack automatically (via lifting).

```lean
def processStrings (inputs : List String) : StateT Int (ExceptT String IO) Unit := do
  for input in inputs do
    tryCatch
      (do let i ← parseInt input; modify (· + i))
      (fun msg => IO.println s!"Skipping '{input}': {msg}")
```

> [!tip] Adding Effects = Adding a Layer
> Need logging? Wrap with `WriterT`. Need configuration? Wrap with `ReaderT`. Each transformer adds exactly one effect without touching existing code.
